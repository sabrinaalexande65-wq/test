Быстрая установка

  • Windows: скачай установщик → запусти → дождись окончания установки.
  • macOS: скачай .dmg → перетащи Brave в Applications → открой Brave.
  • Linux: следуй инструкции на странице Linux (репозиторий + установка через пакетный менеджер).

Официальная справка по установке: см. Brave Support.

Если “не синкается” (Brave Sync)

  • Проверь, что на устройствах одинаковая версия Brave.
  • Попробуй пересоздать цепочку Sync (на одном устройстве создать заново и подключить остальные).
